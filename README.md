# MakeBlack 개인정보처리방침

최종 업데이트: 2026년 4월 14일

## 1. 수집하는 정보

MakeBlack은 서비스 제공을 위해 아래 정보를 수집합니다.

- 이메일 주소 (회원가입 시)
- 이름 및 아이디(핸들)
- 프로필 사진 (선택사항)
- 할 일 및 카테고리 데이터
- 팀 활동 데이터

## 2. 정보 이용 목적

수집한 정보는 다음 목적으로만 사용됩니다.

- 회원 식별 및 로그인
- 할 일 데이터 저장 및 동기화
- 팀 기능 제공

## 3. 정보 보관 및 삭제

- 수집된 정보는 Supabase 서버에 암호화하여 저장됩니다.
- 계정 삭제 시 모든 데이터가 영구 삭제됩니다.
- 법적 의무가 없는 한 제3자에게 정보를 제공하지 않습니다.

## 4. 제3자 서비스

MakeBlack은 아래 제3자 서비스를 사용합니다.

- **Supabase**: 데이터 저장 및 인증 ([개인정보처리방침](https://supabase.com/privacy))
